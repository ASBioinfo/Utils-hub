T-Cell Epitope Analysis

HMC-I prediction comprises two steps: (i) T cell processing and (ii) T cell epitope prediction. T-cell processing step simulates biological process of the input proteins that undergo proteasomal degradation and transport of antigen present(TAP) T-cell epitope predction step calculates binding affinity of each epitope with user's selected MHC alleles.

  • Download T-cell processing program, netCTL1.2b from www.cbs.dtu.dk/services/NetCTL-1.2b/. This program is available in this computer in app folder.
  • Open netCTL file (gedit)in the netCTL1.2b folder and change the "setenv CTLHOME" & "setenv TMPDIR".
  • Open the program folder and run netCTL sh file as "netCTL -s -wt 0.05 -we 1.00 -wc 0.15 >netCTL_out" (for more detail type "man -d ./netCTL.1 ").
  • Select only those line that having with <=E by and convert in fasta format by help of awk command.(<=E depicts potential epitope).
  • Go to IEDB site (https://www.iedb.org/), click MHC-I binding predictions. It opens a MHC-I bindng link in new tab
  • Download the MHC-i tool by clicking "Dowmlaod" of MHC-I binding webpage. This program is available in this computer in app folder by mhc_i name
  • Open mhc_i folder and type "./src/predict_binding.py IEDB_recommended (method name) 9(length of epitope) nonamer_netCTL_fasta(fasta file) >mhc_I_out(output file name).
